# CP1-JOKEMPO (com Card Game Library)

Implementação adicionada:

- Projeto `CardGame.Core` (Class Library)
- Modelo genérico de cartas (Card/Suit/Rank)
- Baralho (`Deck` com Shuffle/Draw)
- `Hand`, `Player`, `Round`, `GameEngine` e `HighestCardGame`
- Documentação de uso e próximos passos

O projeto original Jokempo permanece presente em `Jokempo.Core` e `Jokempo.UI`.
